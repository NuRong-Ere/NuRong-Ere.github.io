安装该安装的

首先我使用的是hexo框架，而在hexo官网上写着，首先需要安装git和nodejs，反正照着安装就是了。

Git

每一个git使用者都绕不开的软件git

只后就是一路next，完成后打开git。在这之后是绑定我们本地仓库和github之间的联系。这里输入你github上的email账号

没有报错成功创建之后就可以进入下一步了
